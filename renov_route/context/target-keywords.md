# Renov'Route — Target Keywords

## Structure des Clusters Thématiques

Chaque cluster est organisé autour de :
- **Mot-clé pilier** : Volume élevé, concurrentiel, cible une page principale
- **Mots-clés de cluster** : 5-10 mots-clés liés, ciblant des sous-pages ou articles
- **Mots-clés longue traîne** : 10-15 variations spécifiques, moins concurrentielles
- **Intention de recherche** : Informationnelle, commerciale, transactionnelle ou navigationnelle
