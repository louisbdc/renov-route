# Rénov Route - Site Web Next.js

Site web pour Rénov Route, spécialiste en marquage et signalisation.

## Technologies

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build

```bash
npm run build
npm start
```
